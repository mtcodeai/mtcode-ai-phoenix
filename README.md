MTCode AI Phoenix is an AI-powered coding assistant designed to help developers write, review, and improve code faster and smarter.

## Features

- **AI Code Generation** – Generate boilerplate and complex code snippets with natural language prompts
- **Code Review** – Get intelligent feedback and suggestions on your code
- **Refactoring Support** – Automatically improve code quality and maintainability
- **Multi-language Support** – Works across popular programming languages

## Getting Started

Clone the repository and follow the setup instructions to get MTCode AI Phoenix running locally.

```bash
git clone https://github.com/mtcodeai/mtcode-ai-phoenix.git
cd mtcode-ai-phoenix
```
